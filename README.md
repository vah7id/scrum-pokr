## Scrum Pokr

Neatest scrum planning poker game.

#### `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


#### `yarn build`

Builds the app for production to the `build` folder.

#### Deployment

`yarn deploy`
